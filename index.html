Neural-network-prostate

Using Artificial Neural Networks to Diagnose Prostate Cancer Malignancy

View on GitHub Download .zip Download .tar.gz

About

This project aims to diagnose Prostate Cancer Malignancy according to the Gleason Grading system with at least an 95% accuracy rate.

Neural Networks

The project uses a Convolutional Neural Network with inputs of 1750 x 1750 pixels and runs through three consecutive layers with 5x5 convolutions and pooling operations. The final layer consists of 14x14 neurons that branch out to 256 inputs. There are four labels, each representing Gleason Non-Cribiform 3, Gleason Non-Cribiform 4, Gleason Cribiform 4, and Normal Prostate as a control sample.

Technical Aspects

Computation is done on CPU due to the lack of resources with the Google Tensorflow Library.
